INTRODUÇÃO AO HTML

Para começar,vamos criar a estrutura básica de um site

Primeiro vamos colocar a tag

<!doctype html>

Isso irá informar o navegador que o site é feito em HTML 5

TAG HTML

A próxima tag é a tag <html>

Tudo que ficar entre <html> e </html> estará dentro do seu site

TAG HEAD

A próxima tag é a tag <head>

Tudo que ficar entre <head> e </head> será a parte comportamental do seu site

TAG BODY

E agora vem a tag <body>

Tudo que estiver entre <body> e </body> será o corpo do seu site

ESSE É O CÓDIGO FUNDAMENTAL DO HTML
